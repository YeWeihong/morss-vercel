🎯 XPath 选择器工具

通过可视化方式选择网页元素，自动生成 XPath 规则，轻松创建自定义 RSS 订阅源

📖 使用说明

方式一：直接加载网页（推荐）

  1. 在下方输入框中输入你想要抓取的网页 URL
  2. 点击"加载网页"按钮
  3. 按照提示，依次点击页面上的元素来选择：
    • 订阅项（items）：所有文章/条目的共同父元素（必选）
    • 标题（title）：文章标题的位置
    • 链接（link）：文章链接的位置
    • 内容（content）：文章摘要或内容
  4. 自动生成的 XPath 规则和 Morss URL 会显示在下方

方式二：手动输入 XPath（适用于 CORS 限制的网站）

  1. 点击"手动输入 XPath"按钮
  2. 在浏览器中打开目标网页，按 F12 打开开发者工具
  3. 使用元素选择器选中页面元素，右键 → 复制 → 复制 XPath
  4. 将 XPath 粘贴到对应的输入框中
  5. 系统会自动优化 XPath 并生成 Morss URL
⚠️ 注意：由于浏览器安全限制（CORS），部分网站可能无法在此工具中直接加载。 对于这些网站，你可以使用浏览器开发者工具（F12）手动获取 XPath，然后使用下方的"手动输入 XPath"功能。 详见《XPath 自定义规则详解》

📝 手动输入 XPath 模式

💡 使用方法：
  1. 在浏览器中打开目标网页，按 F12 打开开发者工具
  2. 点击"元素选择器"（或按 Ctrl+Shift+C）
  3. 点击页面上要抓取的元素
  4. 在开发者工具中，右键点击高亮的 HTML 元素 → 复制 → 复制 XPath
  5. 将复制的 XPath 粘贴到下面对应的输入框中
  6. 系统会自动优化 XPath 并生成 Morss URL

💡 更多帮助

关于 XPath 规则： XPath 是一种在 HTML/XML 文档中查找信息的语言。 本工具会自动生成简洁的 XPath 表达式。

详细文档： 《Morss 自定义 XPath 规则详解》

示例用法：
1. CLI 模式：morss --items="//div[@class='post']" https://example.com
2. Web 模式：https://morss.it/:items=%7C%7Cdiv%5B%40class%3D%27post%27%5D/https://example.com

其他功能： Morss 还支持从 JSON 页面创建订阅源、CSV 导出、缓存等功能。 查看 主页 了解更多。
